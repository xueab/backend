# 情绪日记方法

## 为什么写情绪日记

把情绪写出来本身就是一种整理。很多困扰在脑子里反复打转，一旦落到纸上，往往会显得更清晰、可控。长期来看，情绪日记帮助你：

- 识别自己情绪的**触发点**与**模式**。
- 区分"事实"与"解释"。
- 看到自己的**进步**与**恢复力**，建立对自己的信任。

## 基础模版：五栏记录

每次写 5~10 分钟即可：

1. **时间与情境**：刚才发生了什么？
2. **感受词**：用 1~3 个词描述情绪（如：失落、烦躁、委屈）。
3. **强度（0-10）**：这种情绪有多强烈？
4. **身体感受**：身体哪里有反应？（胸口紧、喉咙堵、头胀）
5. **想法**：当时脑子里闪过的第一句话是什么？

不用追求完整的句子，短语、词组都可以。

## 进阶模版：事实—解释—替代想法

当同一类情绪反复出现时，尝试区分：

- **事实**：客观发生的事（"他没回我消息"）。
- **自动解释**：我当时如何理解（"他是不是生气了"）。
- **其他可能性**：至少列出 2 种另一种解释（"他在忙"、"手机没电"）。
- **如果朋友遇到这事，我会怎么对他说**：通常这时会更温柔、更客观。

## 感恩日记（轻量版）

在低落期间，情绪日记可能让人更关注负面。此时可以搭配"感恩三件事"：

- 今天让我感到一点点温暖的 3 件事（再小都可以，比如"早上喝了热豆浆")。
- 这不是逼自己"正能量"，而是训练大脑同时看见"确实还好"的部分。

## 给身体写一段：身体扫描日记

情绪日记容易让人卡在脑子里。可以偶尔切换到"身体视角"：

- 闭眼花 1 分钟扫描全身，从头到脚。
- 写下：哪里紧、哪里酸、哪里温暖、哪里冷。
- 问自己："如果这个紧张的部位会说话，它想告诉我什么？"
- 把它的话用一两句写下来。

不必神秘化，这是一种把潜意识感受**翻译成语言**的方式。

## 给"未来的自己"写一段

低落时容易觉得"我永远会这样下去"。可以写一封短信给三天后 / 一周后 / 一个月后的自己：

- "亲爱的未来的我，今天我感到 ……"
- "我希望那时的你 ……"
- "我相信你已经 ……"

到了那个日子真的回头看，常常会发现"原来我已经走过来了"，对自我连续感很有帮助。

## 双栏笔记：事实 + 感受

很多人写日记容易写成"评价"。可以试一种简单的双栏：

| 客观发生的事 | 我此刻的感受 / 想法 |
| --- | --- |
| 上午开会被打断 3 次 | 我感到不被尊重，很生气 |
| 中午没吃饭 | 我下午整个人都很烦 |
| 晚上 Ta 没回消息 | 我开始猜想是不是 Ta 不在意 |

把"事实"和"解释"分开写，就能看见**很多情绪其实是被自动解释带起来的**。

## 写完之后的小动作

写完不要"立刻翻篇"，给自己一个收尾：

- 大声读一遍。
- 在最后写一句"接下来我能做的最小一步是 ___"。
- 合上本子，深呼吸 3 次。
- 做一件回到当下的事（喝水、洗手、走两步）。

让日记成为**一次完整的整理**，而不是单纯发泄。

## 几种适合不同状态的日记模板

| 状态 | 推荐写什么 |
| --- | --- |
| 情绪激烈 | 五栏记录（情境 / 感受 / 强度 / 身体 / 想法） |
| 反复反刍 | 事实—解释—替代想法 |
| 长期低落 | 感恩三件事 + 今天我做到了什么 |
| 想复盘冲突 | "我"开头的表达练习（见"人际沟通"一篇） |
| 重大决定前 | 列正反两栏，每栏 5 条 |
| 焦虑高峰 | 担忧清单 + "今晚能做的吗" |

不必固定一种模板，**根据状态选**最合适的那种。

## 使用建议

- **不要追求每天都写**：一周 2~3 次、每次 5 分钟即可，坚持比完美重要。
- **不要把日记当成绝对事实**：写下来的只是"那一刻的你"的想法，事后重读时往往会有新的视角。
- **保护隐私**：日记是给自己的，不是给别人看的。不要为"给别人看"的视角而修饰。
- **如果用电子方式记录**：注意账号安全和云同步设置，避免被亲密关系或同事看到。
- **发现反复出现的负面主题**：如果一段时间内反复写到绝望、自我伤害的内容，请把日记作为**寻求专业帮助的信号**，而不是自我诊断的工具。
