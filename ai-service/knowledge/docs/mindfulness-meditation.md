# 正念冥想入门

## 什么是正念

正念（Mindfulness）是一种"有意识地、不评判地、把注意力放在当下"的能力。它不是清空大脑，也不是追求"什么都不想"，而是允许念头来来去去，自己只做一个温和的观察者。

研究表明，规律的正念练习可以帮助：缓解焦虑与抑郁、改善睡眠、提升专注力、减少情绪化决策。

## 常见误解

- **"我做不到完全不想事情。"** —— 不需要，注意力跑走是正常的，把它温柔地带回来本身就是练习。
- **"必须盘腿打坐很久。"** —— 每天 5 分钟、坐在椅子上同样有效。
- **"必须感觉到平静。"** —— 不一定，有时反而会注意到平时被忽略的烦躁，这也是觉察的一部分。

## 三分钟呼吸空间练习

适合最忙的时候，每次只需 3 分钟：

1. **第一分钟：觉察现在**。问自己——我现在身体什么感觉？情绪是什么？脑里在想什么？只是观察，不评判。
2. **第二分钟：聚焦呼吸**。把注意力放在鼻孔或腹部的呼吸感上，感受一吸一呼。
3. **第三分钟：扩展觉察**。把注意力扩展到整个身体，感受身体在椅子上的支撑、皮肤接触空气的温度。

结束时停一秒，再回到原本的事。

## 身体扫描

适合睡前或情绪过载时做，约 10~15 分钟：

- 平躺，从脚趾开始，把注意力依次移到：脚趾 → 脚掌 → 小腿 → 大腿 → 臀部 → 腹部 → 胸 → 背 → 肩 → 手臂 → 脖子 → 脸 → 头顶。
- 每到一个部位，停留几秒，仅仅"感受"它，不去改变它。
- 如果某处紧绷或有不适，不必逃避，让呼吸"陪伴"它一会儿，再继续。

## 行走冥想

适合不喜欢静坐的人：

- 在一段安静的路上慢慢走 5~10 分钟。
- 把注意力放在脚底接触地面的感觉、腿部肌肉的活动、身体的平衡。
- 不戴耳机、不刷手机，让走路这件事本身成为练习。

## 饮食冥想

把"吃饭"也变成练习的机会：

- 拿起一颗葡萄干或一口饭，先看它、闻它。
- 慢慢咀嚼，感受甜味、咸味、温度、口感的变化。
- 觉察自己想立刻"刷手机"的冲动，但只是让饭这件事本身成为完整的体验。

## 给念头贴标签

冥想时大量念头涌出来是常态，可以学会"贴个标签就放下"：

- 想到工作 → 心里说一句"计划"。
- 想到昨天的争执 → 心里说一句"回忆"。
- 担心明天 → 心里说一句"担忧"。

贴完标签，把注意力轻轻带回呼吸。重点不是消灭念头，而是不被它卷走。

## 建立习惯的小技巧

- **同一时间、同一地点**练，让大脑形成条件反射。
- 用一段固定的引导音频开始，比家庭录音好坚持。
- 每次练习后写一句话："今天注意到了……"，建立反馈感。
- **不要把"少做了一天"当作失败**——明天接着继续就好。

## 边界

正念是普适性的训练，但**不是所有人都适合所有形式**：有重度创伤、急性精神症状或严重抑郁的人，请在专业指导下进行；如果练习中反复出现失控的强烈情绪，请告诉咨询师或医生。
