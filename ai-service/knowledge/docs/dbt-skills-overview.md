# 辩证行为疗法（DBT）核心技能概览

## DBT 适合什么样的困扰

辩证行为疗法（Dialectical Behavior Therapy）最初为情绪剧烈波动、容易冲动行为（自伤、暴食、关系反复）的人设计，今天已被广泛用于：

- 情绪调节困难、容易"一炸就停不下"。
- 长期人际关系紧张、容易把人推开又拉回来。
- 强烈的空虚感、无聊感、不知道自己想要什么。
- 无法忍受痛苦、用极端方式让自己"快点好起来"。

DBT 的核心理念是 **"辩证"**：一方面**接纳此刻的你已经尽力了**，另一方面**也需要改变**。两者同时存在，不矛盾。

## 四大模块

DBT 把技能分成四组，互相支撑：

1. **正念（Mindfulness）**：基础能力，把注意力带回当下。
2. **痛苦耐受（Distress Tolerance）**：在"无法立刻改变"的痛苦中先撑住，不做雪上加霜的决定。
3. **情绪调节（Emotion Regulation）**：识别情绪 + 减少情绪脆弱性 + 改变情绪。
4. **人际效能（Interpersonal Effectiveness）**：在关系中既照顾自己也照顾对方。

下面把每个模块里最常用的几个技巧罗列出来，便于在情绪上头时直接调用。

## 痛苦耐受：TIP 与 ACCEPTS

### TIP（情绪强度 ≥ 8 的"急刹车"）

- **T（Temperature）冷温度**：把脸浸入凉水或敷冰毛巾 30 秒，激活迷走神经，降生理唤醒。
- **I（Intense exercise）剧烈运动**：5~10 分钟跳绳、深蹲、快跑，把肾上腺素消耗掉。
- **P（Paced breathing）节奏呼吸**：呼气长于吸气（吸 4 秒，呼 6~8 秒）。
- **P（Paired muscle relaxation）放松配对**：先用力绷紧肌肉再松开，循环几轮。

### ACCEPTS（注意力转移）

适合无法立刻解决问题、需要先撑过去几小时的场合：

- **A（Activities）做点活动**：散步、整理、做饭。
- **C（Contributing）帮助别人**：哪怕是给家人倒杯水。
- **C（Comparisons）比较视角**：想想自己曾撑过更糟的处境。
- **E（Emotions）切换情绪**：看搞笑视频、听欢快音乐。
- **P（Pushing away）暂时搁置**：把烦恼"装进想象的盒子里"，约定明天再开。
- **T（Thoughts）思维转移**：心里背一首诗、解一道题。
- **S（Sensations）感官刺激**：冰水洗手、闻精油、嚼薄荷糖。

## 情绪调节：PLEASE 与 ABC

### PLEASE（降低情绪脆弱性）

身体不稳定时情绪也会更容易崩。日常照顾好这几条，能显著抬高情绪基线：

- **PL（Physical illness）身体疾病**：及时看医生、按时吃药。
- **E（Eating）饮食**：别让自己饿太久，也别长期暴饮暴食。
- **A（Avoid drugs）回避酒精/毒品**：它们会大幅放大情绪波动。
- **S（Sleep）睡眠**：保证规律睡眠，比任何技巧都重要。
- **E（Exercise）运动**：每周 3 次中等强度。

### ABC（积累正向情绪）

不要等坏事来了才"补救"，而是平时主动充电：

- **A（Accumulate）积累愉快事件**：每天至少做一件让自己愉快的小事。
- **B（Build mastery）建立掌控感**：每周做一件略有挑战、做完会有成就感的事。
- **C（Cope ahead）提前预演**：知道某个场景会让你情绪很大，先在脑里排练应对方式。

## 人际效能：DEAR MAN

清楚地表达诉求，并保持自我尊严：

- **D（Describe）描述事实**：不带评价说出发生了什么。
- **E（Express）表达感受**：用"我感到"开头。
- **A（Assert）明确诉求**：你具体想让对方做什么。
- **R（Reinforce）说明好处**：满足你的诉求对 Ta / 你们关系有什么正面影响。
- **M（Mindful）保持专注**：不被对方带跑题，必要时复读自己的诉求。
- **A（Appear confident）保持坚定**：声音不抖、眼神接触、不为合理诉求过度道歉。
- **N（Negotiate）愿意协商**：理性诉求里，留一定让步空间。

## 一句话辩证

DBT 鼓励在自我对话里**同时持有两面**：

- "我已经尽力了，**而且**我可以学得更好。"
- "我现在很痛苦，**而且**我能做点事情让自己稍微好一点。"
- "我对 Ta 有怨气，**而且**我仍然在乎这段关系。"

把"但是"换成"而且"，是 DBT 的小魔法。

## 边界

DBT 是一个结构化的、长期训练的体系，自助阅读可以入门，但**对中重度的情绪困扰，强烈建议在咨询师指导下系统学习**。如果你正处于反复自伤、强烈自杀念头中，请优先联系本应用"求助渠道"里的专业资源。
